The compiled JS bundle has `http://localhost:18792` hardcoded.
Browser visitors try to hit localhost on their own machine → fails → no data.

**Evidence:** `/_next/static/chunks/app/page-*.js` contains `localhost` + `18792`

**Fix — sAIge action required:**

